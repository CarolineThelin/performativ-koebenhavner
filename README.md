# Performativ Københavner 🧃

En app hvor du kan konkurrere om at være den mest performative københavner.

## Kom i gang
1. Indsæt Figma link i `/design/figma-link.md`
2. Kør `npm install`
3. Kør `npm run dev`

## App Ikon
Eksporter app ikonet fra Figma til `/assets/app-icon` og kør:
```
npx react-native-make set-icon --path ./assets/app-icon/icon.png
```
